# JavaScript Tasks

This repository contains my solutions for the Modern JavaScript tasks from the Udemy learning path.

## Tasks

### Task 1: String Manipulation
- Search for `"paja"` in a string
- Extract every 3rd character and replace `"a"` with `"*"`
- Output in uppercase

### Task 2: Age Validation
- Ask user for age using `prompt` (Node.js uses `readline-sync`)
- Accept age only between 18 and 22
- Output messages depending on the age

### Task 3: Array Operations
- Create array `["Dog", "Horse", "Cow"]`
- Copy array using `map()` and add `["Cat", "Sheep"]`
- Search for elements in the array and handle "not found"

### Task 4: Class Flower
- Define `Flower` class with properties:
  `type`, `color`, `amount`, `inStore`
- Methods to change color, amount, inStore
- Output before and after changes

### Task 5: Current Date
- Get current date
- Format as `Day YYYY-MM-DD` (e.g., Tuesday 2024-01-16)

## How to Run
1. Make sure [Node.js](https://nodejs.org/) is installed
2. Open terminal in the project folder
3. Run tasks using:

```bash
node Task_1.js
node Task_2.js
node Task_3.js
node Task_4.js
node Task_5.js
